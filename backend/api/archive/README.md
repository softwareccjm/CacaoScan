# Archivo Legacy - Scripts de Management Commands

Este directorio contiene scripts de management commands que ya no se usan en el código activo pero se mantienen por referencia histórica.

## Archivos archivados

- `convert_dataset.py` - Script para convertir dataset.csv (legacy, no se usa)
- `map_dataset_images.py` - Script para mapear imágenes al dataset (legacy, no se usa)

## Nota

Estos scripts fueron movidos desde `api/management/commands/` porque no tienen referencias en el código activo. Si necesitas usarlos en el futuro, puedes moverlos de vuelta a `management/commands/`.
